2A_B3208571_周彥廷  (40%)簡報製作、程式製作

題目:踩地雷

1.遊戲進行流程與規則:

遊戲目標：
找出所有沒有地雷的格子，並避開所有地雷。

遊戲規則：
棋盤由網格組成，部分格子藏有地雷。
翻開沒有地雷的格子，會顯示周圍 8 宮格內的地雷數量。
若翻開的格子周圍皆無地雷（顯示為 0），系統會自動展開相鄰的安全區域。
玩家可以對懷疑有地雷的格子「插旗」標記。

勝利條件：翻開所有非地雷的格子。

失敗條件：翻開任何一顆地雷。


2.內容操作說明:

遊戲採命令列（Console / Terminal）文字介面互動。
輸入格式：[動作] [X座標] [Y座標]
動作指令：
r (Reveal)：翻開格子。例如 r 3 4 代表翻開座標 (3,4) 的格子。
f (Flag)：插旗 / 拔旗。例如 f 3 4 代表在 (3,4) 標記地雷。
q (Quit)：中途退出遊戲。

3.程式如何安裝執行:

開發環境：任何支援 C++11 (或以上) 的編譯器（如 GCC, Clang, MSVC）。
編譯指令：打開終端機，移動到程式碼資料夾，輸入以下指令進行多檔案編譯：
g++ main.cpp Game.cpp Board.cpp Cell.cpp -o Minesweeper
執行指令：
Windows: Minesweeper.exe
